💌 Valentine Wall

A simple, wholesome web app where anyone can send and view Valentine’s notes — all displayed on a shared Love Wall.

Originally built as a private project for deployment on Render, this has now been opened up for everyone to explore, use, and build on.

🌹 About

Valentine Wall lets users:

✍️ Write a Valentine’s message
📬 Choose who it’s to
🖊️ Sign who it’s from
💖 See their note added to a public Love Wall

It’s designed to be lightweight, fun, and easy to use — no accounts, no friction, just pure vibes.

🧠 How It Works
A user submits:
Their message 💬
The recipient 💝
The sender ✨
The note gets stored (via the backend)
It’s instantly added to the Love Wall, where everyone can view all submissions
